== Description ==
The Pi Payment Plugin allows WooCommerce store owners to accept payments via the Pi Network. This plugin integrates seamlessly with your WooCommerce store, providing a secure and efficient way to process transactions using Pi cryptocurrency.

== Installation ==
1. Upload the `pi-payment-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under WooCommerce > Settings > Payments.

== Usage ==
Once activated, the Pi Payment option will appear on the WooCommerce checkout page. Customers can select this option to pay using their Pi Network wallet.

== Frequently Asked Questions ==
= How do I generate an API key? =
API keys can be generated through the plugin settings page. Each key is unique to your account and is used to authenticate payment requests.

= Is this plugin secure? =
Yes, the plugin implements various security measures to ensure that all transactions are secure and that sensitive data is protected.
